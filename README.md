## About This Project

A responsive, single-page personal portfolio built with plain **HTML5, CSS3, and Vanilla JavaScript** —
no frameworks, no build step. Styled around a "terminal" visual identity: a fake terminal window in the
hero section, prompt-style section labels, and ASCII-style animated skill bars.

## Features

- Responsive navbar with hamburger menu on mobile
- Terminal-window hero section with typewriter role animation
- Dark/light theme toggle (persisted with localStorage)
- About section with education & quick-info card
- Scroll-triggered animated skill bars (ASCII `[████░░░░]` style, animate once)
- Project grid with tech-tag filtering (all / python / ai / java / cpp / web)
- Contact form with client-side validation (no backend yet — that's Month 2)
- Hidden easter egg: try the Konami code (↑ ↑ ↓ ↓ ← → ← → b a)
